# 參考書籍

我們對命理一直都在摸索階段，過程之中購入不少書籍，也從圖書館借閱了不少書籍。

我們買書通常只是為了節省借還書的時間，就直接在網路下單買了整批書，之後再慢慢看。所以底下的書並不代表就是好書、值得買的書！(命理的書濫竽充數的不少，挑選時請小心！)
